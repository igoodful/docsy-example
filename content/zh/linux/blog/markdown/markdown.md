








## 下划线
> 一般建议加上下划线，会更明确是链接、可点击的
<u>洛阳牡丹</u> **不仅**具有很高的观赏价值，**而且**是名贵的中药材
```html
<u>文本</u>
```

## 加粗
**文本**
```html
**文本**
```

## 斜体
> 前后使用一个星号或下划线

*文本*
```html
*文本*
```
## 删除线
~~文本~~
```html
~~文本~~
```
