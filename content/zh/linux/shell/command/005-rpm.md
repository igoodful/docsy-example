#### 删除软件包

- `-e`或`–erase` 删除软件包
- `–allmatches`  删除所有与PACKAGE_NAME匹配的软件包版本 
- `–nodeps`  防止依赖性检查
- `–noscripts`  阻止命令执行包处理周围的脚本
- `–notriggers`  类似于 –noscripts， 阻止命令触发脚本
- `–test` 检查卸载是否成功或失败，无需实际执行操作


```bash
rpm --test -e PACKAGE_NAME

rpm -e PACKAGE_NAME

rpm --test -e --allmatches --nodeps --noscripts --notriggers  PACKAGE_NAME

```
