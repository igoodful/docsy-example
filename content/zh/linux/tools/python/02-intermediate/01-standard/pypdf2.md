{{< alert >}}

PyPDF2的主要能力在页面级操作，比如：

- 获取PDF文档基本信息

- PDF分割及合并

- PDF的旋转及排序

- PDF加水印及去水印

- PDF加密及解密

{{< /alert >}}

### 安装 pypdf2

```bash

pip install pypdf2

```

### 使用 pypdf2
> PyPDF2的核心两个类是PdfFileReader和PdfFileWriter，完成PDF文件的读写操作。
