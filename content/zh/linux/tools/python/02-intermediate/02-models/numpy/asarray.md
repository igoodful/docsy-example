### 2. 使用


#### 将列表转换为 ndarray
```python
x =  [1,2,3] 
a = np.asarray(x)

# 数据类型转换
x =  [1,2,3] 
a = np.asarray(x, dtype = float)


```


#### 将元组转换为 ndarray

```python
x =  (1,2,3) 
a = np.asarray(x)

```

#### 将元组列表转换为 ndarray

```python
x =  [(1,2,3),(4,5)] 
a = np.asarray(x)

```
