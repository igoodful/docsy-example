## 一、windows

在目录`C:\guolicheng\app\OpenVPN\config\xxx`下面的文件如下：
- ca.crt
- xxx.conf
- xxx.crt
- xxx.key
- xxx.ovpn
- ta.key


#### 1. xxx.conf

```bash
client
dev tun
proto udp
remote 10.10.10.10 1194
resolv-retry infinite
nobind
persist-key
persist-tun
ca ca.crt
cert xxx.crt
key xxx.key
remote-cert-tls server
tls-auth ta.key 1
cipher AES-256-CBC
comp-lzo
verb 3
mute 20
explicit-exit-notify 1
```
#### 2. xxx.ovpn
